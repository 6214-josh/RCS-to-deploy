對測說明

  1. 先確認 AccuPick TCP Server 已啟動，且 RCS 的 rcs.accupick.host/port 正確。
  2. WES 送命令到 RCS 時，可用這頁同步看訂單狀態與 TCP log。
  3. 若看到 COMM_TIMEOUT，通常代表 RCS 沒連到 AccuPick，或 AccuPick 沒回 ACK。
  4. 兩三天後正式對測時，這頁可先拿來做單筆送測與 log 追蹤。
